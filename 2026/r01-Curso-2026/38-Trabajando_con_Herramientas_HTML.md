# 38 - Trabajando con Herramientas HTML

## ¿Qué es un validador de HTML y cómo puede ayudarte a depurar tu código?

HTML es un lenguaje muy indulgente; los elementos se renderizan incluso cuando cometes errores, como olvidar incluir una etiqueta de cierre. Por eso necesitas un validador de HTML.

Existen varios validadores de HTML que puedes utilizar. El más ampliamente aceptado es el servicio de validación de marcas `w3.org`.

Cuando visites el sitio `validator.w3.org`, puedes hacer clic en el botón `Validate by Direct Input` y pegar tu código HTML.

Otro validador de HTML que puedes usar es `jsonformatter.org`.


## Cómo utilizar el inspector de DOM y DevTools para depurar y construir tus proyectos

Para depurar tu código, necesitarás usar algunas herramientas proporcionadas por tu navegador.

Dos herramientas importantes a utilizar serían el inspector de DOM y las herramientas de desarrollador.

El inspector de DOM te permite inspeccionar la estructura HTML de la página en la que estás.

DOM significa Modelo de Objeto del Documento. Es una estructura en forma de árbol que representa los elementos de una página.

Las herramientas de desarrollador te permiten inspeccionar el HTML, CSS y JavaScript de la página en la que estás.
